The ARM64 Debugger's Handbook — Iris, GDB, and Live Debugging on the RD-N2 FVP

“Stop guessing. Attach the debugger, read the registers, follow the evidence.” This is the debugging guide we wish we had when we started bringing up the ARM RD-N2 FVP. Every technique here was learned by needing it — in the middle of a hang, a crash, or a component that refused to speak. Part I: The Debugging Arsenal What is Iris? Iris is ARM’s debug API built into every Fast Model (FVP).
